# Amazon Automation Assignment

## ✅ Scenario Covered
- Search HP Smart Tank
- Select Smart Tank 589
- Add to cart (Qty = 2)

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

- Page Object Model (POM)
- Utility classes for config & waits
- Config-driven test execution

## ▶️ How to Run

2. Open in Eclipse/IDE
3. Run testng.xml
